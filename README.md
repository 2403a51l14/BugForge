# BugForge 🚀

BugForge is an AI-powered code debugging and bug analysis platform designed to help developers identify, analyze, and fix errors in source code efficiently.

## 📌 Project Description

BugForge allows users to:

* Upload or paste source code
* Detect syntax and logical bugs
* Identify the **bug type**
* Suggest **AI-powered fixes**
* Compare original and fixed code
* Analyze **time and space complexity**
* Maintain bug scan history

The project integrates a backend service with AI analysis support to provide fast and intelligent debugging assistance.

## 🛠️ Tech Stack

```bash

```

### 2. Start the Backend

```bash
python server.py
```

### 3. Run the Frontend

Open `index.html` in your browser

OR use VS Code Live Server.
